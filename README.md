# Contract First Development

# Prerequisites
1. AWS CLI `brew install awscli`

# Deployment Steps
1. Set you env variables. See [.env.template](./.env.template) and move it to .env
2. Build `npm run build`
3. Get AWS access keys AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY, AWS_SESSION_TOKEN and set them in your terminal
4. Deploy `npm run deploy`
